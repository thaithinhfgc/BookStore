# BookStore

- ### Both Role Action
     - Login, register new account and logout
     - Change password
     - View and update user information
- ### Customer Role Action
     - Add, delete and update item to cart
     - Create order
     - Check order details
- ### Admin Role Action
     - Create, update, promote and disable user

## Technology

- Frontend
     - JSP
     - Bootstrap
- Backend
     - Servlet
     - SQL Server
